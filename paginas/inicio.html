Bienvenidos al OVI del Curso Programación Web 1

En el presente sitio web podrán encontrar un amplio material de estudio que corresponde a los componentes del Curso Programación Web 1, encontraran información respecto a cómo instalar los dos programas que se utilizaran para el desarrollo del estudio de los temas del curso, además encontraran documentación con los conceptos básicos de base de datos y el lenguaje de programación PHP y una explicación de tallada de cómo hacer la conexión de base de datos MYSQL con PHP. Además, el OVI cuenta con dos videos tutoriales hechos por el creador del sitio (Daniel Pardo) donde se explica cómo crear una base de datos y hacer la conexión con PHP utilizando los dos programas que anteriormente mencionamos.

Objetivos del OVI

Presentar a los estudiantes de la Universidad Nacional Abierta y a Distancia la información y capacitación necesaria para que dichos estudiantes puedan superar todas las competencias que el curso Programación Web 1 demanda, para ellos se implementó cinco documentos informativos con temas del curso y a su vez dos videos tutoriales que explican detalladamente dichos temas en la forma práctica.
